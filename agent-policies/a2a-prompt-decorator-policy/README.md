# A2A Prompt Decorator Policy Example

Allows to inject prompts on every task/send or task/sendSubscribe request such that the admin can inject a new prompt and can put a condition when the prompt will be injected such that it can apply complex behaviour.

## Test the Policy

Test the policy by publishing it to Exchange.
